# Mosque Admin Panel

صفحة إدارة المساجد المضافة من قبل المستخدمين.

## الإعداد

1. افتح ملف `mosques-admin.html`
2. استبدل المتغيرات التالية ببيانات Supabase الخاصة بك:
   ```javascript
   const SUPABASE_URL = 'YOUR_SUPABASE_URL';
   const SUPABASE_ANON_KEY = 'YOUR_SUPABASE_ANON_KEY';
   ```

## الميزات

- ✅ عرض جميع المساجد المضافة
- ✅ إحصائيات (إجمالي المساجد، عدد الأجهزة، المضافة اليوم)
- ✅ بحث في المساجد
- ✅ تعديل بيانات المسجد (الاسم والعنوان)
- ✅ حذف المسجد
- ✅ عرض معرف الجهاز الذي أضاف المسجد
- ✅ تصميم عصري وسريع الاستجابة

## الاستخدام

افتح الملف في المتصفح مباشرة أو ارفعه على Vercel/Netlify.
